## hw5
文件目录结构：
* hw5即为项目目录
* hw5目录内存放有录制演示视频


设计思路：
每一个disk根据不同的颜色配给不同的差异速度和不同的分数。各种颜色随机出现。出现的位置和方向随机决定。在达到指定的分数之后分数清空，进入下一轮。
